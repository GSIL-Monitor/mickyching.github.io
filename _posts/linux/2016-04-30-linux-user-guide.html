软件源配置

grub配置

  • Grub2/Setup

如果要让启动菜单不包括其他磁盘的系统信息，可以使用如下配置。 

# file: /etc/default/grub
GRUB_DISABLE_OS_PROBER=true

配置完成之后运行命令更新grub配置。 

sudo update-grub

fstab配置

如果要让系统启动之后自动挂载磁盘，可以加入如下配置。 

# file: /etc/fstab
UUID=ab1b3396-4d52-4baf-9c0c-b497d30d8df2 /repo           ext4    defaults        0       2
UUID=1c4e18b9-9b9b-4ef5-8e90-b06da97e16e9 /data           ext4    defaults        0       2

输入法配置

安装fcitx需要卸载ibus输入法，卸载ibus输入法时注意不要卸载ibus，因为卸载ibus会导致整个unity桌面环境被卸载。 

sudo apt-get purge ibus-gtk ibus-gtk3 ibus-table
sudo apt-get install fcitx fcitx-googlepinyin

安装完fcitx之后需要运行如下命令选择输入法。 

im-config

flash插件

由于谷歌的开源浏览器将不再支持Netscape浏览器插件API，Adobe公司的Flash将无法正常工作。用户可以使用Pepper Flash Player，这是谷歌浏览器上一款替代Adobe Flash的插件。 

sudo apt-get install pepperflashplugin-nonfree
sudo update-pepperflashplugin-nonfree --install

如果是ubuntu早期版本，需添加源： 

sudo add-apt-repository ppa:skunk/pepper-flash
sudo apt-get update
sudo apt-get install pepflashplugin-installer
sudo update-pepperflashplugin-nonfree --install

安装jekyll
  • http://blog.csdn.net/cschengvdn/article/details/42425271
  • http://www.linuxidc.com/Linux/2014-09/106771.htm

如果用apt-get安装jekyll失败，则可以用gem安装，首先需要修改gem源。 

gem sources -l
gem sources --remove https://rubygems.org/
gem sources -a https://ruby.taobao.org/

然后利用gem安装： 

sudo apt-get install ruby1.9.1-dev
sudo gem install jekyll v 2.5.3
sudo gem install therubyracer
sudo apt-get install curl
curl --silent --location https://deb.nodesource.com/setup_0.12 | sudo bash -
sudo apt-get install --yes nodejs build-essential
